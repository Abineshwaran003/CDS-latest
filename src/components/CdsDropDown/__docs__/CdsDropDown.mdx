# CDS-DROPDOWN

A dropdown component that supports ID-label formatted options.

## Usage

```tsx
import CdsDropDown from "../CdsDropDown";

const options = [
  { id: "1", label: "Apple" },
  { id: "2", label: "Banana" },
  { id: "3", label: "Cherry" },
];

<CdsDropDown label="Select a fruit" options={options} size: "small" />;  // Default

<CdsDropDown label="Select a fruit" options={options} disabled size: "medium" />;  // disabled

<CdsDropDown label="Select a fruit" multiselect = {true} options={options} size:"large" />;  // Multiselect

```
