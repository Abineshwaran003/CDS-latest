# TitleBar

TitleBar component with different props.

#### Example

<Canvas of={TitleBar.Primary} />

## Usage

```ts
import {TitleBar} from "cds";

const Example = () => {
  return (
      <TitleBar
        version="MES v2.0"
      home
      notification
      settings
      info
      profilePicture="https://via.placeholder.com/40"
      profileUserName="Jane Doe"
      profileRole="Manager"
      homeOnClick={()=> console.log("Clicked")}
      settingsOnCLick={()=> console.log("Clicked")}
      moduleOnclick={()=> console.log("Clicked")}
      profileLogoutOnClick={()=> console.log("Clicked")}
      />
  );
};

export default Example;
```

#### Arguments

- **text** _`() => void`_ - A string that represents the text content of the TitleBar.
- **primary** - A boolean indicating whether the TitleBar should have a primary styling or not. Typically, a primary TitleBar stands out as the main action in a user interface.
- **disabled** - A boolean indicating whether the TitleBar should be disabled or not. When disabled, the TitleBar cannot be clicked or interacted with.
- **size** - A string with one of three possible values: "small," "medium," or "large." It defines the size or dimensions of the TitleBar.
- **onClick** - A function that is called when the TitleBar is clicked. It receives a MouseEventHandler for handling the click event on the TitleBar element.
